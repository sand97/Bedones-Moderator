**How**: Configure in Group Settings > Admin Tools > Welcome Message.

### 2. Admin Validation for Links and Media

**Enable Restrictions**:
- Group Settings > Permissions
- "Only admins can send messages" OR
- "Approve new members + monitor 1st week"

**Advantage**: Filters 90% of scams before propagation.

**Case Avoided - Entrepreneurs Group Douala (320 members)**:
- Admin validation enabled
- 12-15 scam link sharing attempts/month blocked
- 0 victims in 8 months vs similar groups (5-10 victims/year)

### 3. Active and Fast Moderation

**Recommended Timeframe**:
- Suspicious links: Deletion < 5 min
- Spam: < 15 min
- Hate speech: IMMEDIATE

**Tools**:
- Notifications enabled (even at night if active group)
- Trusted co-admins (2-3 people)
- Monitoring 2-3 times/day minimum

### 4. Drastically Limit External Links

**Strict Policy**:
- Links authorized ONLY if:
  - Known/reliable source
  - Admin verified destination
  - Relevant to group

**Automatic Blacklist**:
- Shortened links (bit.ly, tinyurl) â†’ Systematic deletion
- Suspicious domains (.tk, .ml, .ga)
- URLs with "promo", "free", "win"

**Alternative**: Create information channel where only admins post verified links.

### 5. New Member Verification

**Before Accepting**:
- Real profile photo?
- Known/recommended number?
- If unknown: Ask validation question ("How did you find this group?")

**Red Flags**:
- Empty/generic profile
- Foreign number not justified
- Addition request without context

**Stat**: 68% of scammers have profiles created < 30 days.

### 6. Immediately Exclude Repeat Offenders

**Tolerance Policy**:
- 1st spam/suspicious link: Public warning
- 2nd time: Temporary exclusion 48h
- 3rd time: Permanent exclusion + blocking

**Important**: Apply rule fairly (even friends/family).

### 7. Preserve Moderation Evidence

**Document** (screenshots):
- Deleted messages (with date/time)
- Excluded members (with reason)
- Warnings given

**Storage**: Google Drive/iCloud secure, 12 months minimum.

**Usefulness**: If complaint or investigation, proof of diligence = legal protection.

**Suggested Format**:
```
[Date] - [Time] - [Member Name] - [Action] - [Reason]
15/01/26 - 14h23 - Jean K. - Message deleted - Investment scam link
```

### 8. Educate Members Regularly

**Monthly Reminders**:
"Security reminder: NEVER send money to strangers via this group. Always verify identity before transaction. Admins NEVER ask for money."

**Awareness of Common Scams**:
- Fake jobs "urgent hiring high salary"
- Fake packages "customs fees to pay"
- Fake contests "you won"

## What to Do if a Member Threatens You Legally

### Step 1: Stay Calm and Courteous

**DO NOT**:
- Delete evidence
- Panic and erase everything
- Immediately exclude person

**DO**:
- Capture all exchanges
- Contact lawyer if serious threat
- Gather proof of diligence

### Step 2: Prove Your Good Faith

Show:
- Group rules published from start
- Documented moderation actions
- Responsiveness to problematic content
- Attempts at amicable resolution

### Step 3: Mediation Before Judicial

Propose private discussion to resolve dispute. 80% of cases settle this way.
